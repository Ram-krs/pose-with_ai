Pose With AI - Project Documentation

Project Name
            Pose With AI

Project Title
            AI-Based Smart Pose Detection and Photography Assistant

Objective
            Develop an AI-powered camera assistant that detects body posture, provides pose suggestions,
analyzes pose quality, and helps users capture better photos.
Problem Statement
Many users struggle to find suitable poses for photography. This project provides real-time AI
guidance to improve posture and photo quality.

Frontend
            HTML, CSS, JavaScript, MediaPipe Pose

Backend
            Python, Flask

Database
            Local Storage (Current Version), SQLite/MongoDB (Future Enhancement)

How It Works:

1. User opens application.
2. Camera detects body landmarks.
3. AI analyzes pose.
4. Score is calculated.
5. Suggestions are displayed.
6. User captures photo manually.

Key Concepts
            Computer Vision, AI, Pose Estimation, Real-Time Processing, Body Landmark Detection, Pose
Scoring.
